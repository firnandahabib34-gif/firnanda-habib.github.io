Tentang Saya

Saya adalah mahasiswa Teknik Informatika di Politeknik Negeri Batam dan juga bekerja sebagai teknisi printer dan komputer. Saya memiliki minat besar di bidang web development dan pembuatan aplikasi berbasis web. Saat ini saya sedang mengembangkan aplikasi web pengelolaan rapat sebagai salah satu proyek kuliah saya.

Proyek Saya

Aplikasi web untuk membantu proses pencatatan dan pengelolaan rapat di kampus. Dikembangkan menggunakan HTML, CSS, dan JavaScript dengan konsep CRUD sederhana.
